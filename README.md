# AgentSpace Development Dashboard

Public static dashboard for tracking development of the private AgentSpace
platform.

## How it works

- `data/checklist.json` contains the historical roadmap and task display.
- `data/status.json` contains only the current public verified projection.
- `data/status.json` is generated by the private AgentSpace verification workflow
  and must not be edited manually.
- `app.js` loads both data files independently and renders their public fields
  with safe DOM APIs.
- `index.html` provides the page structure.
- `styles.css` controls presentation and responsive behavior.

This public repository has no private-repository credentials or access. If a
publication fails, the last published status remains available. The dashboard
marks verified status as stale after seven days without an update, and a failure
to load one data source does not prevent the other source from rendering.
